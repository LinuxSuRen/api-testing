`atest` 发布 `v0.0.20`

`atest` 是致力于为开发者提供一站式接口开发、测试的开源工具，无需注册账号、完全私有化部署、可扩展、可定制。

* Mock 服务中增加 TCP 类型的代理 (#692) @LinuxSuRen
* 支持多线程下载插件 (#669) @LinuxSuRen
* 支持通过 HTTP 协议启动插件 (#664) @LinuxSuRen
* 支持 SQL 语言编辑器的自动补全 (#722) @LinuxSuRen
